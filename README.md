This is a vanilla web app built with [Parcel](https://parceljs.org)

## Getting Started

Run the dev server:

```bash
yarn start
# or
npm start
```

And open [localhost:1234](http://localhost:1234) in your browser!

or
use the vercel link to directly open it in your browser
